Run this SQL in your Supabase dashboard → SQL Editor to create the required table: 

CREATE TABLE IF NOT EXISTS tll_schedules (
  id             uuid        DEFAULT gen_random_uuid() PRIMARY KEY,
  title          text        NOT NULL,
  youtube_url    text        NOT NULL,
  start_time     integer     DEFAULT 0,
  end_time       integer     NOT NULL,
  scheduled_date date        NOT NULL,          -- e.g. 2025-04-20
  streaming_time text        NOT NULL,          -- e.g. 09:30:00
  is_played      boolean     DEFAULT false,
  played_at      timestamptz,
  is_active      boolean     DEFAULT true,
  created_at     timestamptz DEFAULT now()
);

-- Allow anonymous read/write (disable RLS or add policies)
ALTER TABLE tll_schedules ENABLE ROW LEVEL SECURITY;

CREATE POLICY "public read"  ON tll_schedules FOR SELECT USING (true);
CREATE POLICY "public write" ON tll_schedules FOR ALL    USING (true) WITH CHECK (true);

After creating the table, refresh this page. The player at tlltv.html will automatically pull today's overrides and fall back to the default schedule for any unscheduled time slots.
